# FlyingCarNanoDegree
Udacity Flying Car and Autonomous Flight Engineer Nanodegree Program

Backyard project - Python backyard_flyer.py 25 5 -- will make the drone fly in a square 25 units wide and 5 units high (defaults to 20 and 3 if no input args are specified). 
A matplotlib plot will open to display the path traced by the drone.
